# Szablony Image Promptów (Nano Banana)

Prompty do generowania obrazów dla Nano Banana 2.5 / Pro.

---

## Podstawowa struktura

```
[Główny podmiot] + [Działanie/Poza] + [Lokalizacja/Tło] + [Oświetlenie] + [Styl]
```

Przykład:
```
Młoda kobieta w czerwonej sukience tańczy na plaży
o zachodzie słońca w stylu impresjonizmu
```

---

## Szablony według kategorii

### Portret

```
[Typ portretu] [opis osoby] w [lokalizacja].
[Oświetlenie]. [Wyraz twarzy/nastrój].
[Styl opcjonalnie].
```

**Przykłady:**

```
Zbliżenie na twarz starszego mężczyzny z siwą brodą.
Ciepłe światło zachodzącego słońca. Zamyślony wyraz twarzy.
```

```
Portret do pasa młodej kobiety w białej bluzce.
Profesjonalne oświetlenie studyjne. Pewny siebie uśmiech.
Tło szare gradientowe.
```

```
Dziecko bawiące się w parku jesienią.
Naturalne światło dzienne. Radość i energia.
```

---

### Krajobraz

```
[Typ krajobrazu] [pora dnia/roku].
[Elementy charakterystyczne].
[Atmosfera/nastrój].
[Styl opcjonalnie].
```

**Przykłady:**

```
Górski krajobraz o świcie wiosną.
Śnieżne szczyty, zielone doliny, mgła w dolinach.
Spokój i majestat natury.
```

```
Tropikalna plaża o zachodzie słońca.
Palmy, turkusowa woda, różowo-pomarańczowe niebo.
Rajski, relaksujący nastrój.
```

```
Miejski krajobraz Tokio nocą w deszczu.
Neonowe światła odbijające się w mokrym asfalcie.
Atmosfera cyberpunk.
```

### Produkt / E-commerce

```
[Produkt] na [tło/powierzchnia].
[Oświetlenie]. [Kąt ujęcia].
[Dodatkowe elementy kontekstowe].
```

**Przykłady:**

```
Elegancki zegarek na skórzanym pasku na ciemnym drewnie.
Profesjonalne oświetlenie studyjne z delikatnymi refleksami.
Ujęcie pod kątem 45 stopni.
```

```
Butelka perfum na białym marmurze.
Miękkie światło z lewej strony. Minimalistyczne tło.
Obok kilka płatków róży.
```

```
Nowoczesne słuchawki bezprzewodowe unoszące się w powietrzu.
Gradientowe tło od niebieskiego do fioletowego.
Dynamiczne ujęcie.
```

### Ilustracja / Concept Art

```
[Scena/Koncept] w stylu [styl artystyczny].
[Kluczowe elementy]. [Kolorystyka].
[Nastrój/atmosfera].
```

**Przykłady:**

```
Zamek w chmurach w stylu Studio Ghibli.
Pływające wyspy, wodospady spadające w niebo.
Pastelowe kolory, bajkowa atmosfera.
```

```
Futurystyczne miasto na Marsie w stylu sci-fi concept art.
Kopuły habitatów, czerwony pustynny krajobraz.
Dramatyczne oświetlenie dwóch słońc.
```

```
Magiczny las w stylu dark fantasy.
Świecące grzyby, mgła, stare drzewa z twarzami.
Tajemnicza, lekko niepokojąca atmosfera.
```

### Jedzenie / Food Photography

```
[Danie/Jedzenie] na [talerz/podłoże].
[Oświetlenie]. [Styl fotografii].
[Elementy dekoracyjne].
```

**Przykłady:**

```
Soczysty burger z frytkami na drewnianej desce.
Ciepłe światło boczne. Styl rustykalny.
Obok szklanka piwa i świeże warzywa.
```

```
Elegancki deser czekoladowy na białym talerzu.
Profesjonalne oświetlenie z góry. Styl fine dining.
Dekoracja z malinami i listkiem mięty.
```

```
Świeże sushi na czarnym kamieniu.
Minimalistyczne tło. Naturalne światło.
Pałeczki i miska sosu sojowego obok.
```

## Edycja istniejących obrazów

### Zmiana tła

```
[Obraz wejściowy]

Zmień tło na [nowe tło].
Zachowaj [co zachować] bez zmian.
```

Przykład:
```
[Zdjęcie osoby]

Zmień tło na panoramę Nowego Jorku o zmierzchu.
Zachowaj osobę i jej oświetlenie.
```

---

### Zmiana stylu

```
[Obraz wejściowy]

Przekształć w styl [nazwa stylu].
Zachowaj kompozycję i główne elementy.
```

Przykład:
```
[Zdjęcie zwykłe]

Przekształć w styl pop art jak Andy Warhol.
Jaskrawe kolory, kontrastowe cienie.
```

---

### Dodawanie elementów

```
[Obraz wejściowy]

Dodaj [element] w [lokalizacja w obrazie].
[Styl elementu ma pasować do reszty].
```

Przykład:
```
[Zdjęcie pokoju]

Dodaj kota śpiącego na kanapie.
Kot powinien pasować do oświetlenia pokoju.
```

---

### Usuwanie elementów

```
[Obraz wejściowy]

Usuń [element do usunięcia].
Uzupełnij tło w sposób naturalny.
```

---

## Multi-image fusion

```
[Obraz 1: opis roli] [Obraz 2: opis roli] [Obraz 3: opis roli]

Połącz:
- Z obrazu 1 weź [co]
- Z obrazu 2 weź [co]
- Z obrazu 3 weź [co]

Wynik: [opis końcowego obrazu]
```

Przykład:
```
[Zdjęcie osoby] [Zdjęcie tła górskiego] [Zdjęcie stroju]

Połącz:
- Twarz i postawę z obrazu 1
- Tło z obrazu 2
- Strój turystyczny z obrazu 3

Wynik: Ta osoba w stroju turystycznym na szczycie góry.
```

## Checklist dla promptów obrazowych

- [ ] Główny podmiot jasno określony
- [ ] Lokalizacja/tło opisane
- [ ] Oświetlenie/pora dnia (jeśli istotne)
- [ ] Styl artystyczny (jeśli inny niż fotorealizm)
- [ ] Nastrój/atmosfera
- [ ] Naturalny język (nie parametry techniczne)
- [ ] Dla edycji: jasne co zmienić, co zachować
