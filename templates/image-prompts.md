# Nano Banana 2.5

Prompty do generowania obrazów dla Nano Banana 2.5 / Pro.

---

## Podstawowa struktura

```
[Główny podmiot] + [Działanie/Poza] + [Lokalizacja/Tło] + [Oświetlenie] + [Styl]
```

Przykład:
```
Młoda kobieta w czerwonej sukience tańczy na plaży
o zachodzie słońca w stylu impresjonizmu
```

---

## Szablony według kategorii

### Portret

```
[Typ portretu] [opis osoby] w [lokalizacja].
[Oświetlenie]. [Wyraz twarzy/nastrój].
[Styl opcjonalnie].
```

**Przykłady:**

```
Zbliżenie na twarz starszego mężczyzny z siwą brodą.
Ciepłe światło zachodzącego słońca. Zamyślony wyraz twarzy.
```

```
Portret do pasa młodej kobiety w białej bluzce.
Profesjonalne oświetlenie studyjne. Pewny siebie uśmiech.
Tło szare gradientowe.
```

```
Dziecko bawiące się w parku jesienią.
Naturalne światło dzienne. Radość i energia.
```

---

### Krajobraz

```
[Typ krajobrazu] [pora dnia/roku].
[Elementy charakterystyczne].
[Atmosfera/nastrój].
[Styl opcjonalnie].
```

**Przykłady:**

```
Górski krajobraz o świcie wiosną.
Śnieżne szczyty, zielone doliny, mgła w dolinach.
Spokój i majestat natury.
```

```
Tropikalna plaża o zachodzie słońca.
Palmy, turkusowa woda, różowo-pomarańczowe niebo.
Rajski, relaksujący nastrój.
```

```
Miejski krajobraz Tokio nocą w deszczu.
Neonowe światła odbijające się w mokrym asfalcie.
Atmosfera cyberpunk.
```

---


```
Nowoczesne słuchawki bezprzewodowe unoszące się w powietrzu.

Dynamiczne ujęcie.
```

## Edycja istniejących obrazów

### Zmiana tła

```
[Obraz wejściowy]

Zmień tło na [nowe tło].
Zachowaj [co zachować] bez zmian.
```

Przykład:
```
[Zdjęcie osoby]

Zmień tło na panoramę Nowego Jorku o zmierzchu.
Zachowaj osobę i jej oświetlenie.
```

---

### Zmiana stylu

```
[Obraz wejściowy]

Przekształć w styl [nazwa stylu].
Zachowaj kompozycję i główne elementy.
```

Przykład:
```
[Zdjęcie zwykłe]

Przekształć w styl pop art jak Andy Warhol.
Jaskrawe kolory, kontrastowe cienie.
```

---

### Dodawanie elementów

```
[Obraz wejściowy]

Dodaj [element] w [lokalizacja w obrazie].
[Styl elementu ma pasować do reszty].
```

Przykład:
```
[Zdjęcie pokoju]

Dodaj kota śpiącego na kanapie.
Kot powinien pasować do oświetlenia pokoju.
```

---

### Usuwanie elementów

```
[Obraz wejściowy]

Usuń [element do usunięcia].
Uzupełnij tło w sposób naturalny.
```

---

## Multi-image fusion

```
[Obraz 1: opis roli] [Obraz 2: opis roli] [Obraz 3: opis roli]

Połącz:
- Z obrazu 1 weź [co]
- Z obrazu 2 weź [co]
- Z obrazu 3 weź [co]

Wynik: [opis końcowego obrazu]
```

Przykład:
```
[Zdjęcie osoby] [Zdjęcie tła górskiego] [Zdjęcie stroju]

Połącz:
- Twarz i postawę z obrazu 1
- Tło z obrazu 2
- Strój turystyczny z obrazu 3

Wynik: Ta osoba w stroju turystycznym na szczycie góry.
```

## Checklist dla promptów Nano Banana

- [ ] Główny podmiot jasno określony
- [ ] Lokalizacja/tło opisane
- [ ] Oświetlenie/pora dnia (jeśli istotne)

# Grok Aurora (Obrazy)

Prompty do generowania obrazów dla Grok Aurora (model `grok-2-image`).

**Kluczowe różnice vs Nano Banana:**
- Architektura autoregressive (nie diffusion)
- Preferuje język fotograficzny/filmowy
- Optymalna długość: 600-700 znaków
- Lepszy w tekście i logo

---

## Podstawowa struktura

```
[Podmiot] + [Styl] + [Nastrój/Atmosfera] + [Oświetlenie] + [Detale techniczne]
```

Przykład:
```
A confident businesswoman in a modern glass office,
professional headshot, natural window lighting, sharp focus,
corporate style, neutral background
```

---

## Szablony według kategorii

### Portret profesjonalny

```
[Typ ujęcia] of [opis osoby],
[lokalizacja/tło], [oświetlenie],
[fokus], [styl], [dodatkowe detale]
```

**Przykłady:**

```
Professional headshot of a confident middle-aged businessman,
modern office background with glass walls, natural window lighting
from the left, sharp focus on eyes, shallow depth of field,
corporate style, neutral color palette
```

```
Close-up portrait of a young creative director,
exposed brick studio background, dramatic rim lighting,
high contrast, editorial style, black and white
```

```
Environmental portrait of a chef in restaurant kitchen,
stainless steel background, warm ambient lighting,
medium shot, documentary style, authentic moment
```

- [ ] Podmiot na początku promptu (subject-first)
- [ ] Używam języka fotograficznego (bokeh, wide-angle, etc.)

# Grok Imagine (Wideo)

Prompty do generowania krótkich klipów wideo (6-15 sekund) dla Grok Imagine.

**Kluczowe informacje:**
- Długość: 6-15 sekund
- FPS: 24
- Natywna synchronizacja audio-wideo
- Tryby: Normal, Fun, Custom, Spicy

---

## Podstawowa struktura

```
[Podmiot + Ruch] + [Tło + Ruch] + [Kamera + Ruch] + [Styl] + [Atmosfera]
```

Przykład:
```
A lone figure walking through a rainy cyberpunk alley,
neon signs reflecting on wet pavement, slow camera pan right,
cinematic mood, night scene
```

---

## Szablony według kategorii

### Akcja / Ruch

```
[Osoba/Obiekt] [akcja w czasie ciągłym],
[tło z elementami ruchu], [ruch kamery],
[tempo], [styl], [atmosfera]
```

**Przykłady:**

```
A skateboarder performing a kickflip in slow motion,
empty concrete skatepark at golden hour, camera tracking
the movement, dust particles in backlight, urban sports vibe
```

```
Runner sprinting through city streets at dawn,
buildings passing in background blur, tracking shot,
real-time speed, Nike commercial style, determined energy
```

```
Dancer spinning gracefully in abandoned warehouse,
dust particles floating in light beams, slow orbit camera,
slow motion, artistic documentary style, ethereal mood
```

---

### Atmosfera / Ambient

```
[Element] [subtelny ruch],
[otoczenie], [ujęcie kamery],
[nastrój], [styl]
```

**Przykłady:**

```
Candle flame flickering gently in dark room, soft warm light
dancing on nearby objects, static close-up shot, peaceful
and meditative mood, ASMR aesthetic
```

```
Rain drops hitting window glass,
city lights blurred in background, macro static shot,
melancholic atmosphere, lo-fi aesthetic
```

```
Steam rising from hot coffee cup,
cozy morning kitchen setting, slow push in,
warm and inviting mood, lifestyle commercial style
```

---

### Loop / Seamless

```
[Element z ruchem cyklicznym],
[tło statyczne lub cykliczne], [ujęcie],
[styl], [przeznaczenie: seamless loop]
```

**Przykłady:**

```
Ocean waves gently rolling onto sandy beach, seamless loop,
bird's eye view, turquoise water, white foam patterns,
relaxing tropical atmosphere
```

```
Neon sign flickering in endless loop,
dark urban alley background, static medium shot,
cyberpunk aesthetic, perfect loop
```

```
Fireplace flames dancing continuously,
cozy living room setting, static wide shot,
warm hygge atmosphere, seamless loop for ambient display
```

---

### Cinematic / Storytelling

```
[Postać] [akcja narracyjna],
[otoczenie budujące historię], [ruch kamery filmowy],
[styl filmowy], [nastrój emocjonalny]
```

**Przykłady:**

```
A detective walking down rain-soaked noir street,
neon signs and steam rising from grates, slow dolly forward,
1940s film noir style, mysterious and tense atmosphere
```

```
Astronaut stepping onto alien planet surface,
strange rock formations and two suns in sky, epic wide shot
slowly zooming out, sci-fi blockbuster style, awe and wonder
```

```
Child running through wheat field towards sunset,
golden stalks swaying in wind, tracking shot from behind,
Terrence Malick style, nostalgic and dreamy mood
```

---

## Typy ruchu kamery

| Ruch | Opis | Użycie |
|------|------|--------|
| Static | Nieruchoma kamera | Ambient, loops |
| Pan | Obrót w poziomie | Panoramy, śledzenie |
| Tilt | Obrót w pionie | Ujawnianie, dramatyzm |
| Dolly | Ruch do przodu/tyłu | Zbliżanie, oddalanie |
| Tracking | Podążanie za podmiotem | Akcja, sport |
| Orbit | Okrążanie podmiotu | Produkty, portrety |
| Crane | Ruch w górę/dół | Epickie ujęcia |

---

## Typy tempa

| Tempo | Użycie |
|-------|--------|
| Slow motion | Dramatyzm, detale, sport |
| Real-time | Naturalność, dokumenty |
| Timelapse | Upływ czasu, chmury, miasto |
| Speed ramp | Efekty specjalne, transitions |

---

## Checklist dla promptów Grok Imagine

- [ ] Ruch podmiotu jest explicite opisany
- [ ] Ruch tła jest określony (lub "static")
- [ ] Ruch kamery jest określony
- [ ] Długość pasuje do 6-15s (jeden koncept)
- [ ] Proste ruchy (unikam złożonych choreografii)
- [ ] Unikam szczegółowych opisów rąk
- [ ] Unikam tekstu w wideo
- [ ] Wybrany odpowiedni tryb (Normal/Fun/Custom)
- [ ] Dla loops: zaznaczam "seamless loop"
